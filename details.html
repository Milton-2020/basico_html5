Uso de la etiqueta DETAILS

permite definir contenido que el usuario pueda visualizar u ocultar

Dicha etiqueta cuenta con un nico atributo OPEN este atributo es de tipo booleano signific que si se especifica se muestra visible el contenido de la etiqueta details si no se especifica aparece oculto

Por otra parte para que se visualize como titulo de la etiqueta, sirviendose asu ves de texto de referencia, para visualizar o no su contenido se usa la etiqueta summary
