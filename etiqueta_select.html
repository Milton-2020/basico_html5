Uso de la etiqueta select

Etiqueta select.

Permite la creación de lista desplegables o listas abiertas, desde la que podemos realizar la selección de una o varias de las opciones que se presentan en la misma.

La etiqueta se configura en conjunto con otra etiqueta denominada option, la cual permite especificar las opciones que aparecen en la lista.

Atributos:

autofocus: permite especificar cual es el control que recibe el "foco", cuando se realiza la carga del documento web en el navegador. Es un atributo booleano y solo con aparecer significa que esta activo.

disabled: permite especificar que el elemento sobre el cual el que se aplica, se ha de desactivar. Un elemento desactivado está bloqueado y no se puede cambiar ni realizar clic sobre el mismo.

form: permite especificar a qué formulario pertenece el campo que se está solicitando especificando el atributo identificador del formulario. Como valor se ha de especificar el valor del atributo id, asociado al formulario donde se encuantra incluido la etiqueta.

multiple: permite especificar en campos de tipo select. Que se puedan seleccionar una o más opciones de la lista que se presenta.

name: permite especificar el nombre que se quiere asignar al campo de formulario.

required: el atributo aplicado a un campo permite especificar que la introducción de datos en el campo ha de ser obligatorio, es decir no se puede dejar en blanco.

size: el atributo permite especificar un valor numérico, el cual corresponderá con las opciones visibles que se presentan de la lista en el campo.

por otra parte la etiqueta para especificar las opciones depende de la etiqueta "option"

Etiqueta option.

permite especificar las opciones de la lista creada con la etiqueta select.

Atributos:

disabled: especifica que el elemeto sobre se aplica, se ha de desactivar. Un elemento desactivado está bloqueado y no se puede cambiar ni realizar clic sobre el mismo.

label: permite especificar una etiqueta corta en formato de texto, para la opción que se está especificando.

selected: permite especificar que la opción sobre la cual se aplica, es la opción preseleccionado cuando el documento es cargado en el navagador.

value: el atributo especifica el valor que se quiere asignar a la opción que se presenta en la lista de forma que este valor es el enviado, cuando se pulsa el botón de envió del formulario.

Etiqueta optgroup.

la cual pemite especificar grupos de opciones, agrupando las opciones comunes por grupos. Esta última etiqueta solo dispone de dos atributo.

Atributos:

label: el cual permite especificar el valor del nombre del grupo.

disabled: pemite especificar que el grupo este activado o desactivado.
