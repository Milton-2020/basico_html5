Uso de la etiqueta Video

la etiqueta video permite integrar y poder reproducir un video en el documento web

Algunos atributos que componen la etiqueta video

Atributo SRC: este atributo permite definir la ruta en la cual se encuantra el archivo de video

Atributo AUTOPLAY: especifica que el archivo de video se reprodusca automaticamente al cargar la pagina

Atributo COTROLS: permite visualixar los controles de control de video como lo son play, volumebe, pantalla completa, etc

Atributo LOOP: permite generar un bucle que se repite una y otra ves el archivo de video

Atributo MUTED: con este atributo al momento de cargar el archivo de video se inicializa en modo silencio

Atributo PRELOAD: este atributo curnta con tres posobles valores, none, auto y metadata

none: no hace ninguna precarga del archivo de video y espera asta que el usuario inice la reproduccion

auto: si se define este valor se inicializa la precarga del archivo de video

metadata: precarga solo datos del archivo de video,duracioin, dimenciones, etc

Atributo HEIGHT: permite definir el alto del video expresado el valor en pixeles

Atributo WIDTH: permite definir el ancho del video expresado en pixeles

Etiqueta SOURCEpermite la carga del achivo de video en varios formatos y cuenta con dos atributos src y type

Atributo src: permite definir la ruta ya sea absoluta o relativa de donde esta alojado el archivo de video

Atributo type: se usa para indicar el tipo MIME del archivo que se quiere reproducir
