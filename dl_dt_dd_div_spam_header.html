Uso de las etiquetas DL, DT, DD, DIV, SPAM, HEADER

Uso de la etiqueta Dl

Permite definir lo que se le denomina lista de definicion, la cual a su bes esta compuesta por termino y un definicon

Para realizar este tipo de lista se usan dos etiquetas mas, dt y dd.

DT: permite definir el termino

DD perite especificar la definicion

Uso de la etiqueta div y spam

Dichas etiquetas no tiene ningun tipo de significado especial, lo do que div se define como un elemento en bloque un genera un salto de linea antes y despues, ocupa el todo el ancho del documento web

mientras que spam es un elemento en linea, solamente ocupa el espacio ajustado al contenido

Uso de la etiqueta Header

esta etiqueta se incluye dentro de la denominada web semantica que consiste en la utilizacion de estas etiquetas para la estructura del documento

aun que no es obligatorio el uso de la misma si es recomendado, ya que los navegadores dan importancia a la estructura semantica

esta etiqueta se puede usar dentro de otras etiquetas siempre y cuando no sea head o footer
