Uso de la etiqueta AUDIO y sus atributos

La etiqueta de audio como su nombre lo indica nos permite integrar contenido de audio

Atributo src: nos permite definit la ruta absoluta o relativa de donde esta alojado el archivo de audio

Atributo Autoplay: permite que al cargar la pagina se reprodusca de manera automatica el audio

Atributo Controls: muetar los controladores basico de un reproductor como lo es pausar, play, control de volumen, etc.

Atributo Loop: este atributo es de tipo booleano y permite crear un bucle en cual se retipa una y otra ves el audio

Atributo Muted: permite que el reproductor se inicie en silencio es de tipo booleano

Atributo Preload: este atributo permite precargar el contenidoo del audio para que no se genere cuelques o pausas a la hora de reproducir el audio.

cuenta con tres posibles valores: none no se realiza nongun tipo de precarga. Auto: se realiza pa precarga completa del audio, este valor se a de usar si no se compromete la velocidad entre la respuesta de cliente servidor

por ultomo si definimos el valor metadata: solo se precargara informacion correspondiente al archivo de audio como lo puede ser duracion, dimencion, etc

Etiqueta Source: permite realizar la carga del archivo de audio en varios formatos cuenta con dos atributos src y type

Atributo type: este atributo se utiliza para especificar el tipo MIME del archivo de audio
