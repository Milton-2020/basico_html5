Uso de DFN, BDO, PRE


La etiqueta BDO define la direccion de visualizacion del texto

cuenta con un atributo dir que puede tener dos valores ltr ( left to right) o rtl (right to left)

el siguiente "Texto" esta escrito de derecha a izquierda

el siguiente "Texto" esta escrito de izquierda a derecha

La etiqueta DFN se usa para dar una definicion de un termino, el navegador lo visualiza en cusiva

BIOS

Sistema de entrada y salida de un sistema informatico

La etiqueta PRE permite definir lo que denominamos texto con formato o preformatiado

al usar esta etiqueta el formato del texto se mostrara tal como se escriba en la pagina web
