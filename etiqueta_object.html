Uso de la etiqueta OBJECT y sus atributos

Etiqueta OBJECT

Se utiliza para incertar elementos multimedia como audio, video, applets de java, activex, pdf y flash

Tambien es posible utilizar la etiqueta para incertar otra pagina web, dentro del propio documento.

Y si se quisiera pasar parámetros al objeto incrustados se podria utilizar la etiqueta param que permite utilizar esta accion

atributos usandos por la etiqueta:

Atributo WIDTH: defiene el ancho que a de tener el contenedor del objeto que se quiere incertar, expresando el valor en pixeles

Atributo HEIGTH: define el alto que a de tener el contenedor del objeto que se quiere incertar, expresado el valor en pixeles

Atributo DATA: permite especificar la URL del recurso al cual se quiere acceder, siendo en este caso el archivo que contiene el objeto, animacion etc.

Atributo NAME: define el nombre que se quiere aplicar al objeto incertado, para de esta forma poder referenciarlo posteriormente, si se usa un lenguaje de cliente

Atributo TYPE: se uasa para espesoficar el tipo MIME del documento con el que se esta realizando el enlace. se puede consultar en la siguiente direccion

MIME
