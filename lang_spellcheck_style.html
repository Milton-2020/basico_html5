Uso de las atributos globales lang, spellcheck y style


Uso del atributo lang este atributo permite indicar al nevegador el idioma en el cual esta escrito el contenido de esa etiqueta html

this is a text in english

ceci est un texte en français

este es un texto en español

Enlace: codigo de lenguajes


Uso del atributo spellchech, este atributo permite que se aplique por parte del navegaro corrección ortografica, que tenga el elemento sobre el cual se aplica

Nota: Comun mente se aplica en input, textarea entre otros elementos de entrada

en el caso de input que no sean contraseñas

en el caso de textarea que contenga el atributo contenteditable

el atributo Spellchech tiene dos valores true o false

Este parrafo tiene activacion de correccion ortografica activada

Este parrafo no tiene activada la correccion ortografica


Uso del atributo style, este atributo implimenta estilos CSS

Estes estilo es global es decir que tiene preferencia sobre cualquier otro estilo, aplicado mediante la etiqueta style o desdes una hoja de estilo externa

Este parrafo tiene un estilo de color asociado

Este parrafo tiene un estilo de fuente asociado

Este parrafo implemanta un estilo de fuente y un color asociado
