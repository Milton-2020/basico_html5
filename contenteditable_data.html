Uso de los atributos globales contenteditable y data


El atributo contenteditable como su nombre lo indica permite editar entre lineas, comun mente se suele usar en parrafos, areas de texto etc

dicha atributo cuenta con 4 valores el varlo true(verdadero), false(falso), inherit(heredado) y ningun valor

Este parrafo no de puede editar por que tiene el valor false

Este parrafo si se puede editar por que tiene el valor true

Este parrafo si se puede editar por que no tiene ningun valor el atributo



El atributo data-* se usa para almasenar datos privados que puede almacenar un documento u aplicasion web, este tipo de datos lugo puedese ser usado para lenguajes de programacion como javaScrpt
