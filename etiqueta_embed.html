Uso de la etiqueta EMBED y su satributos

La Etiqueta EMBED:

permite definir un contenedor para incertar una aplicación exrterna o un contenido interactivo.

Esta etiqueta no requiere de etiqueta de cierre

Atributos de la etiqueta

Atributo SRC: permite definir la url del recurso al cual se quiere acceder siendo en este caso el archivo que contiene el objeto, animacion, etc

Atributo TYPE: sirve para especificar el tipo MIME del documento con el que se esta realizando el enlace

Atrbuto WIDTH: permite definir el ancho que a de contenedor del objeto que se quiere incertar, expresandoe el valor en pixeles

Atributo HEIGTH: permite definir el alto que a de tener el contenedor del objeto que se quiere incertar, expresando el valor en pixeles
