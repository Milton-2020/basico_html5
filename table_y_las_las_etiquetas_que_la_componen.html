Uso de la etiqueta table, caption, th, tr, td, caption, colspan, border


Uso de la etiqueta TABLE

dicha etiqueta permite definir lo que es un atabla compuesta por filas y columnas

Uso de la etiqueta TH

Define la cabesera de una tabla

Uso de la etiqueta TR

Define las filas en una tabla

Uso de la etiqueta TD

Define la celda en una tabla

Uso de la etiqueta CAPTION

Define el nombre o titulo de la tabla

Uso de la etiqueta COLGROP

Especifica un grupo de uno o mas columnas para aplicar formato

Uso de la etiqueta COL

Especifica las propiedades de una columna de las columnas definidas en un elemnto COLGROP

Uso de la etiquete THEAD

Define la cabecera de la tabla

Uso de la etiqueta TBODY

Define el cuerpo de la tabla

Uso de la etiquea TFOOT

Define el pie de la tabla

NOTA: el atributo border se aplica comun mente a al etiqueta TABLE y solo tiene dos valores numericos 1 o 0, aun que lo recomendable es darle formato desde una hoja de estilo

la etiqueta COLSPAN premite unir una o mas celdas se aplica a la etiqueta TD
