Uso de la etiqueta Button

Etiqueta Button

Permite definir un borton sobre el cual se realizan pulsaciones de raton. Si la etiqueta se usa dentro de un formulario practicamente se puede usar los mismos atributos de la etiqueta input.

Aunque se recomienda que para integrar botnes en un formulaio se use la etiqueta input.

Por otra parte la utilizacion de la etiqueta button fuera de un formulario de suele usar par ala ejecucion de alguna funcion en JavaScript. La diferencia con los botones de un formulario, es que con esta etiqueta se puede utilizar en el contenido de la etiqueta texto o imágenes.

Se recomienda que se use el atributo type de la etiqueta para especificar el valor button, ya que dependiendo del navegador el tipo por defecto para la etiqueta se puede interpretar de diferente forma.
