Uso de las siguientes etiquetas cite, i y s

la etiqueta cite se usa para resaltar el texto donde se mencione un titulo o el autor, nombre se un libro o de una pelicula

Nintendo creo el juego de Zelda


La etiqueta i se puede usar para resaltar una frase en otro idioma o un termino en especifico, anteriro mente se usaba como italica

esto es un saludo en ingles Hello


la etiqueta S se usa para tachar texto, indicando que no es correcto, no es exacto o no es permitido

se pronuncia dinero no pisto
