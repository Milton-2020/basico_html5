Uso de la etiqueta IMG permite definir una imagen en el documento

tecnicamente la imagen no se inserta en la pagina web sino se genera un en lace a la misma y el navegador se encarga de situalrla en el documento web

dentro de los formatos mas usados se puede usar Jpeg, png entre otros mas. Se a de tener encuenta que el tama침o de la imagen influye en la carga del documento web, por lo tanto se suele preparar la imagen para reducir su peso

Nota: esta estiqueta no cuenta con etiqueta de cierre


Uso del atributo SRC

dicho atributo permite especificar el origen de la imagen o ruta de donde esta alojada la imagen, mediante una ruta absoluta o relativa

Uso del atributo HEIGTH

Este atributo perimte definir el alto de la imagen en pixeles, siempre que se especifique el ancho y el alto el navegador reservara ese espacio correspondiente para la imagen

si no se espesifica el tamlo de la imagen el navegador no reserbara ese espacio por lo tanto esperara ala carga completa de la imagen

Uso del atributo WIDTH

dicho atributo permite especificar el ancho de la imagen con un valor numerico en pixeles

Uso del atributo ALT

permite definir un texto aleatorio para la imagen, algunos navegadores visualizan este texto como una peque침a ventana emergente

este mensaje se muestra cuando se hay algun problema con el atributo src o por la conexion muy lenta

Uso del atributo ISMAP

dicho atributo se usa para, indicar que la imagen tiene zonas seleccionable que sirven como enlaces

el atributo solo puede existir si la etiqueta img es decendiente de una etiqueta enlace, el atributo ismap es un booleano es decir no tiene valor asociado

si aparece en una etiqueta img indica que es un mapa de imagen

Uso del atributo USEMAP

permite especificar que la imagen utiliza un mapa de imagen con zonas seleccionables asociado a etiqueta con un mapa de imagen definido en la etiqueta map

el atributo usemap no es permitido si la etiqueta img es decendiente de un enlace o bot칩n

la etiqueta map permite definir zonas rectangulares, circulares o poligonales
