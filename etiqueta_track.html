Uso de la etiqueta TRACK y sus atributos

Dicha etiqueta nos permite poder integrar subtítulos (pistas de texto) a los elementos de audio y video, para realizar la visualización esta etiqueta requiere inclusión de un archivo con la extensión .vtt

Atributos de la etiqueta TRACK

Atributo SRC: este atributo permite especificar la direccion URL del recurso al cual se quiere acceder, en este caso a un archivo .vtt

Atributo DEFAULT: permite definir cual es el archivo por defecto en caso aque existieran varios archivos de subtítulos, es de tipo booleano sigfnifica que con solo especificarlo es aplicado

Atributo LABEL: permite especificar entre comillas el titulo para la pista de texto.

Atributo KIND y sus valores

Valores del atributo kind
Valor Descriptión
captions Defiene la traducción de díalogo y efectos de sonido. (Aptos para personas con discapadidad uditiva)
chapters Define títulos de capítulos. Recomendable para navegación por recursos multimedia.
descriptions Define una descripción textual del video que ese esté reproduciendo.
subtitles Define subtítulos. Utilizado para visualizar los subtítulos en un video.

Atributo SRCLANG: este atributo define en su valor el idioma en el cual se encuentra el subtítulo, utilizando los codigos del lenguaje.
El atributo es obligatorio si se implementa el atributo kind y el valor subtitles

Como crear un archivo de subtítulos con extencion .vtt

Al ser un archivo de texto plano es muy facil de crear, en la primera linea del documento deve de inicar WEBVTT, luego un salto de linea y se define un CUES

Un CUES es como un bloque dentro del archivo, que esta compuesto por un id, un rango de tiempo inicio y fin y el texto que sera el titulo del video.

El formato de tiempo es el siguiente:

HH:MM:SS:MMM --> HH:MM:SS:MMM
