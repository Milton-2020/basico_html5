Uso de las etiquetas U, Small, Sub


La etiqueta U se usa para subrayado texto, se usa para definir texto estilistocamente diferente, o nombres en otro idioma

este texto con tiene un parrafo que no se podria interpretar correctamente


La etiqueta Small nos permite generar texto en letra pequeña

este texto destaca por su tamaño


La etiqueta Sub nos permite usar un sub indice por debajo de la base

la formula H2O se puede visulizar de esa forma gracias a la etiqueta sub


La etiqueta sup realiza un formato de superindice, en el labehador se visualiza por encima de la linea base

el texto a con tininuacion muesta la linea base y el sulperíndice
