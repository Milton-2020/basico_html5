Uso de la etiqueta meter

Etiqueta meter.

Permite definir un control que reprecenta una medida escalar dentro de un rango conocido. Se suele para expresar uso de disco, importancia de un resultado, etc.

No de recomienda su utilización para expresar un indicador o barra de progreso, utilizándose para esa tarea la etiqueta progress.

Aun que se puede especificar un contenido de texto en la etiqueta, este por lo que se ha observado no se tiene efecto visual en la reprecentación en el navegador.

Atributos:

form: el atributo buto permite especificar a que formulario pertenece el campo que se esta solicitando especiificando el atributo identificador del formulario. como valor se a de identificar el var del atributo id, asociando al formulario donde se encuentra incluida la etiqueta.

value: al tributo permite especificar el valor de la barra indicadora (es el valor incial que tendra la barra de progreso).

low, high, optimun: los atributos permiten especificar los valores numéricos a partir de los cuales se concideran que son valores bajos y altos respectivamente. Asi como el valor considerado como optimo dentro del rango. Al indicar como valor de la barra indicadora un valor comprendido entre los rangos bajos y alto, se cambia el color de visualizacion de la misma.

min y max: los atributos permiten especificar los valores numéricos considerados mínimos y máximos respectivamente de la barra indicadora.
