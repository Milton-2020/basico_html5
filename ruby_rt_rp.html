Uso de las etiquetas RUBY, RT y RP


La etiqueta RUBY se uas para generar notaciones en formato ruby, que se usa par tipografia de Asia Oriental

permite mostrar la prununciacion de carateres de este Asiatico

dicha etiqueta esta conpuesta por dos etiquetas mas RT y RP

La etiqueta RT permite realizar la explicasion o pronunciacion del texto

La etiqueta RP permite visualisar un texto en los navegadores


Las siguientes siglas significan PC(Orange Juise)
