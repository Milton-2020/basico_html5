Uso de la etiqueta iframe

Etiqueta iframe.

Permire la incerción de un marco en línea, dentro del documento web. Un marco permite la incerción de un documento como puede ser otro documento web, un mapa de google maps, etc.

Atributos

width: permite especificar el ancho en pixeles.

height: permite especificar el alton en pixeles.

name: permite especificar el nombre que se le quiere asignar al marco incertado en el documento web.

src: el atributo permite especificarel origen del objeto que se quiere incertar en el marco, es decir su ubicasión, bien mediante una ruta absoluta o relativa.

srdoc: permite especificar el contenido del objeto en línea, es desir si se especifica contenido en este atributo y el navegador soporta el mismo, se sobreescribe el contenido del atributo src, precentánose el contenido especificado en este atriburo.

seamless: el atributo permite especificar que el marco se precente sin barras de desplasamiento ni bordes, de esta forma su integración dentro del documento web, es mas suave. Es un atributo booleano, para activarlo solo ha de estar precente en la etiqueta.

sandbox: el atributo permite especificar ciertas restricciones para el contenido del marco, siempre y cuando se utilize el atributo. Si se especifica el atributo con un valor vacío(""), se aplican todas las restricciones, en otro caso se an de especificar las restricciones separadas por espacio en blanco dentro de la cadena de texto que se utiliza como valor del atributo.

Valores del atributo sandbox.

Valor del atributo Descripción
"" Se aplican todas las restrucciones.
allow-same-origin El contenido del marco se trata de la misma forma que el
documento web que lo contiene.
allow-top-navigation Permite que el contenido del marco, pueda interactuar con el
contenido del documento web que lo contiene en la
"navegación" por el mismo.
allow-forms Permite que el contenido del marco pueda realizar envió de
formularios.
allow-script Permite que el contenido del marco pueda ejecutar script.
